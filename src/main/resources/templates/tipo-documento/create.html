Nuevo Tipo de Documento

Formato
- Utilice ceros (0) para indicar la posición donde debería ir un número.
- Utilice letras 'a' para indicar la posición donde debería ir una letra.
- Utilice un guión (-) para indicar la posición donde debería ir un guión.

Ejemplos:
DUI: 00000000-0
NIT: 0000-000000-000-0
